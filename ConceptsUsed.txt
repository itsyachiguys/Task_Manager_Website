JAVASCRIPT : 

Basic Syntax and Structure
Variable declarations: const, let
Function declarations: function name() {}
Arrow functions: () => {}
Object literals: { key: value }
Array literals: []
String literals: "text", 'text'
Template literals: `${variable}`
Comments: // single-line, /* multi-line */

Data Types and Values
Primitive types: strings, numbers, booleans
Objects: {} with properties
Arrays: [] with elements
null and undefined (implicitly used)
NaN (not explicitly, but in calculations)

Operators
Assignment: =
Arithmetic: +, -, *, /, %
Comparison: ===, !==, >, <, >=, <=
Logical: &&, ||, !
Ternary: condition ? true : false
Spread/rest: ... (not used, but in array methods)
Increment/decrement: ++, -- (not directly, but in loops)

Control Flow
Conditional statements: if, else if, else
Loops: forEach (array method), implicit loops in setInterval
Switch statements: switch, case, break, default
Return statements: return

Functions and Scope
Function calls: functionName()
Parameters and arguments
Global scope: window object
Local scope: variables inside functions
Closures (implicit in event listeners and intervals)

DOM Manipulation
Element selection: document.getElementById(), document.createElement()
Element properties: .innerHTML, .textContent, .className, .classList
Event handling: addEventListener(), onclick
Form handling: preventDefault(), submit event
Dynamic content: appending elements with appendChild()

Browser APIs
LocalStorage: localStorage.getItem(), localStorage.setItem()
Date objects: new Date(), date arithmetic
Math object: Math.floor(), Math.random(), Math.ceil()
Timers: setInterval()
Window object: window.onload, window.App

Array Methods
push(): adding elements
splice(): removing elements
indexOf(): finding index
forEach(): iterating
filter(): filtering elements
sort(): sorting
slice(): extracting subarrays

String Methods
trim(): removing whitespace
split() (not used, but in JSON operations)
String concatenation: +

JSON Handling
JSON.parse(): parsing strings to objects
JSON.stringify(): converting objects to strings

Error Handling
Try-catch (not explicitly used, but in date calculations)

ES6+ Features
Destructuring (not used)
Modules (not used, but window.App for sharing)
Promises/async (not used)

Other Tiny Concepts
Property access: .property, [index]
Method calls: object.method()
Boolean coercion: in conditionals
Type coercion: in string concatenation
Event object: e in event handlers
This context: not used, but in functions
Immediately Invoked Function Expressions (IIFE): not used
Prototypes: not used
Regular expressions: not used
